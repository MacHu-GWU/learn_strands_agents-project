Install
------------------------------------------------------------------------------

``learn_strands_agents`` is released on PyPI, so all you need is to:

.. code-block:: console

    $ pip install learn-strands-agents

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade learn-strands-agents
